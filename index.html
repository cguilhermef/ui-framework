Tipography

Tipography são classes para adequação de textos.

.fwb

Font weight bold.
Exemplo:
Lorem ipsum dolor sit amet consectetur. 
  Lorem ipsum dolor <span class="fwb"> sit amet</span> consectetur.

.fwl

Font weight light.
Exemplo:
Lorem ipsum dolor sit amet consectetur. 
  Lorem ipsum dolor <span class="fwl"> sit amet</span> consectetur.

.fsi

Font style italic.
Exemplo:
Lorem ipsum dolor sit amet consectetur. 
  Lorem ipsum dolor <span class="fwl"> sit amet</span> consectetur.
